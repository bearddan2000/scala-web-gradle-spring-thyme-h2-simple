# scala-web-gradle-spring-thyme-h2-simple

## Description
A thyme springboot scala gradle build,
that connects to in memory h2 database.

## Tech stack
- springboot
  - thymeleaf
- gradle
  - h2 drivers
  - lombok
  - PostConstruct annotation
- bootstrap
- jquery
- dataTable

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
